## Installation

### Using NPM

```bash
npm install @portkey-v1/request
```

### Using Yarn

```bash
yarn add @portkey-v1/request
```

## Prerequisites

- :gear: [NodeJS](https://nodejs.org/) (LTS/Fermium)
- :toolbox: [Yarn](https://yarnpkg.com/)/[Lerna](https://lerna.js.org/)
